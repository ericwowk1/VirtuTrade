# VirtuTrade

A modern paper trading platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- User authentication with NextAuth.js
- Real-time paper trading simulation
- Portfolio management
- Trading tournaments
- Educational resources

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- NextAuth.js
